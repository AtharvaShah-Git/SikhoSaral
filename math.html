Mathematics Study Materials

1. Algebra Basics

Algebra is a branch of mathematics dealing with symbols and the rules for manipulating these symbols. Key concepts include:

  • Variables: Symbols representing numbers (x, y, z)
  • Constants: Fixed numerical values
  • Expressions: Combinations of variables and constants (2x + 3)
  • Equations: Statements of equality (2x + 3 = 7)

Solving linear equations: ax + b = c

Steps: 1) Isolate variable term, 2) Divide by coefficient

Example: 2x + 3 = 7 → 2x = 4 → x = 2

2. Trigonometry Fundamentals

Trigonometry deals with relationships between angles and sides of triangles. Basic functions:

  • Sine (sin): Opposite/Hypotenuse
  • Cosine (cos): Adjacent/Hypotenuse
  • Tangent (tan): Opposite/Adjacent

Key identities:

  • sin²θ + cos²θ = 1
  • tanθ = sinθ/cosθ

Common angles: sin30° = 0.5, cos60° = 0.5

3. Introduction to Calculus

Calculus studies change and has two main branches:

  • Differential Calculus: Rates of change (slopes)
  • Integral Calculus: Areas and accumulation

Basic derivative rules:

  • Power rule: d/dx(xⁿ) = nxⁿ⁻¹
  • Constant rule: d/dx(k) = 0

Example: If f(x) = x², then f'(x) = 2x
