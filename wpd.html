2. CSS Fundamentals

CSS styles HTML elements:

  • Selectors: Target elements
  • Properties: Style attributes
  • Values: Style specifications

Example:

3. Layout Design

Arranging page elements:

  • Box Model: Margin, Padding, Border
  • Positioning: Static, Relative, Absolute
  • Flexbox: Flexible layouts

Example:

                .container {
                    display: flex;
                    gap: 20px;
                }
