# SikhoSaral

![HTML](https://img.shields.io/badge/HTML5-E34F26?style=for-the-badge&logo=html5&logoColor=white)
![CSS](https://img.shields.io/badge/CSS3-1572B6?style=for-the-badge&logo=css3&logoColor=white)
![License](https://img.shields.io/badge/License-MIT-green?style=for-the-badge)

SikhoSaral is a simple educational website designed for **1st-year Computer Engineering students**. It provides an easy-to-navigate platform for accessing study resources, subject-wise notes, quizzes, and basic student interaction pages such as login, registration, feedback, and contact.

## Features

- Modern and clean landing page
- Study resources section

## Getting Started

### Clone the repository

```bash
git clone https://github.com/AtharvaShah-Git/SikhoSaral.git
```

### Navigate to the project

```bash
cd SikhoSaral
```

### Run the project

Simply open **index.html** in your preferred web browser.

### Optional: Run using a local server

```bash
python3 -m http.server 8000
```

Then visit:

```
http://localhost:8000
```
