Programming in C Study Materials

1. Basic Structure

C is a structured programming language:

  • #include - Header file
  • main() - Entry point
  • Data types: int, float, char

Basic program:

                #include 
                int main() {
                    printf("Hello World");
                    return 0;
                }

2. Control Structures

Decision making and loops:

  • if-else: Conditional execution
  • for: Counter-controlled loop
  • while: Condition-controlled loop

Example:

                if (x > 0) {
                    printf("Positive");
                }

3. Functions

Reusable code blocks:

  • Declaration: return_type name()
  • Definition: Actual code
  • Call: Using function name

Example:

                int add(int a, int b) {
                    return a + b;
                }
